# AI 写作基础规范 (AI Writing Standard)

## 1. 核心理念 (Core Philosophy)

本文件定义了所有通过 AI 生成的内容都必须遵循的通用、基础性原则。它是项目内所有具体“写作档案 (Writing Profile)”的基石。任何特定任务的写作规范都可以扩展或覆盖此处的规则，但绝不能违背其核心理念。

**最终目标：** 创造知识导向、价值驱动、对用户负责的高质量内容。

## 2. 通用核心原则 (Universal Core Principles)

这些原则适用于所有类型的写作任务，无论其具体目标或格式如何。

*   **价值优先，产品其次 (Value First, Product Second):**
    *   **绝对禁止营销语言:** 内容必须是知识导向和价值驱动的。严禁任何形式的广告、导购、夸大或主观的营销术语。
    *   **产品角色降级:** 当需要提及特定产品时，它应仅作为解决问题的**示例**或贯穿文章的**线索**，自然地融入内容，绝不能成为推销对象。

*   **对用户负责 (User Responsibility):**
    *   **内容梯度:** 内容必须对“无门槛用户”友好，使用清晰易懂的语言解释复杂概念，同时为专业人士提供可验证的深度价值。
    *   **信息源标准:** 优先使用学术期刊、官方报告、顶级研究机构数据等权威来源。谨慎使用行业媒体（需交叉验证）。严禁使用论坛、软文等不可靠信源。

*   **原创性与质量 (Originality and Quality):**
    *   **高度原创:** 最终输出的内容必须是高度原创的，不能是简单地替换同义词或对原文进行小幅改写。
    *   **逻辑严谨:** 所有论点都必须有清晰的逻辑链条和可靠的数据/事实支撑。

## 3. 全局格式与风格约束 (Global Format & Style Constraints)

*   **默认语言 (Default Language):**
    *   除非特定“写作档案”中另有规定，所有最终交付的文章正文**必须使用纯正、地道、流畅的英文**。
    *   本规范及其他规范文档本身，使用中文编写。

*   **禁用格式 (Forbidden Formats):**
    *   严禁在内容中使用 `FAQ` (问答)、`Q&A`、自问自答或测试题等形式。
    *   严禁在标题中使用 `Review` 字样。

*   **清单 (Checklist) 要求:**
    *   清单（或列表）中的项目必须是完整的陈述句，而非疑问句或短语。

*   **图片格式 (Image Format):**
    *   所有图片必须使用标准的 Markdown 格式插入：`![图片描述](图片链接)`。图片描述应简洁、清晰，并具有说明性。

## 4. “写作档案” (Writing Profiles)

本项目采用“基础规范 + 写作档案”的模式。
*   **基础规范 (本文):** 提供全局性的、不可违背的准则。
*   **写作档案 (Writing Profiles):** 针对某一特定任务类型（如“SEO文章修订”、“技术规格撰写”）的详细、可执行的指南。每个档案都默认继承本基础规范，并可以定义其独特的角色、工作流、约束和输出格式。
