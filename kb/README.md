# AI 知识库 (AI Knowledge Base)

## 1. 目的 (Purpose)

本目录是 CodeCraftAI 项目的“第二大脑”，旨在构建一个动态的、自增长的知识网络。其核心目标是实现 AI 在特定主题领域的**知识复利**。

每一次写作任务都不应是孤立的。通过将每次任务后学到的核心知识、数据和洞见结构化地存储在这里，AI 在未来的任务中将能够：

*   **加速研究过程:** 在启动新任务时，首先从内部知识库中检索已有信息，而不是每次都从零开始。
*   **提升洞察深度:** 基于已有的知识，AI 能够更快地发现信源之间的联系、差异和矛盾，从而提炼出更深刻的原创性综合观点。
*   **保证观点一致性:** 在创作一系列相关主题的文章时，能够保持核心论点和数据的一致性。

## 2. 工作原理 (How It Works)

本知识库与 `specs/ai-writing-standard.md` 中定义的核心工作流深度集成。

1.  **任务开始前 -> 检索:** AI 在研究任何主题之前，**必须**首先检索本知识库，查找相关的现有知识笔记。
2.  **任务完成后 -> 贡献:** AI 在完成一篇文章后，**必须**遵循 `specs/knowledge-contribution-guide.md` 的规范，进行复盘，并将新的、有价值的知识贡献回本库中。

## 3. 知识笔记格式 (Note Format)

为了确保知识的可查询性和可维护性，所有知识笔记都**必须**使用 `kb/template.md` 模板进行创建。该模板使用 YAML frontmatter 来定义关键的元数据，使得每一份知识都成为一个结构化的数据单元。
