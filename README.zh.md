# CodeCraft AI 开发框架

欢迎使用 CodeCraft，这是一个结构化的、文档驱动的、用于专业 AI 软件开发的框架。本项目提供了一个完整的生态系统，包括工具、工作流和安全功能，以确保 AI 代理能够以可预测、安全和高效的方式为您的代码库做出贡献。

该框架建立在 **“文档优先”** 的原则之上：所有流程、标准和协议都在文档中明确定义，该文档是人类和 AI 开发者的最终事实来源。

## 核心特性

- **规范驱动的工作流**: 一种强制性的开发流程，要求在编写任何代码之前都必须有详细的规范 (`spec`)。
- **CLI 自动化**: 一个强大的命令行界面 (`cli.py`)，用于管理从任务创建到完成的整个开发生命周期。
- **AI 安全协议**: 一套多层安全系统，包括用于安全文件删除的 `.trash/` 目录和用于防止意外修改关键代码的受保护文件清单。
- **内置质量保证**: 集成了带有 `black`（用于格式化）和 `ruff`（用于代码检查）的 `pre-commit` 钩子，从一开始就确保代码质量。
- **国际化**: 所有核心文档均提供完整的双语支持（英文和中文）。

## 核心概念

- **任务 (Tasks)**: 工作单元，在 `.md` 文件中定义，并在 `tasks/` 目录中管理。
- **规范 (Specs)**: 为每个任务创建的详细技术设计，在 `specs/` 目录中管理。
- **架构决策记录 (ADRs)**: 用于记录重要架构选择的文档，在 `adr/` 目录中管理。

这种项目结构强制实现了清晰的关注点分离，并确保每一次变更都经过充分的文档记录和深思熟虑的实施。
